## 核心概念先搞清楚：CN2 GIA、CMIN2、Tier 1 是什么

**这是选套餐之前必须理解的东西**，搞错了买了也白买。

**CN2 GIA（Premium 系列）**：中国电信的高端国际专线，GIA 是 Global Internet Access 的缩写。双向走这条线路，延迟低、丢包少，是目前国内用户连美国 VPS 质量最好的路由。三网的意思是电信、联通、移动三家运营商去程和回程都走这套优化线路。电信用户体验最好，联通移动也有专门的高端路由对应。晚高峰实测从国内到洛杉矶机房延迟大概在 155-165ms 左右，丢包率接近零。

**CMIN2 / 9929（Eyeball 系列）**：电信联通走 9929（联通精品线路），移动走 CMIN2（移动高端国际出口）。没有 CN2 GIA 那么顶，但对联通和移动用户来说往往体验更好，价格也便宜不少。

**Tier 1（国际线路）**：接入国际一级运营商，没有专门针对中国大陆的路由优化。如果你用 VPS 主要是面向海外用户、跑北美业务、或者对国内回程没要求，选这条最划算。超量流量后不关机，只是限速继续跑，对流量需求大的场景很友好。

一句话总结：电信用户追求最好速度选 **Pro（CN2 GIA）**；联通移动用户性价比选 **Eyeball**；不需要国内优化、追求大带宽或大流量选 **Tier 1**。

**电信用户，追求最好的线路质量**

选 LAX.Pro。电信去程走 CN2 GIA，回程也是 GIA，晚高峰延迟稳定，丢包率极低。预算有限就从 WEE 年付套餐（$36.9/年）进，摸一下感觉，实际用起来做代理完全够用。
